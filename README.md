# Analisis-Financiero
Proyecto Final 

# Descripción
Este proyecto incluye:
- Análisis de Google Colab
- Dashboard en Power BI

# ¿Qué se realizó?
Se procesaron datos de diferentes sectores, para el caso de Phyton utilizamos datos sobre la salud de varios países y como una enfermedad principalmnete afectaba a estos países. En Power BI procesamos datos de ventas de una empresa estadounidense distribuida por todo el mundo, cpn el fin de conocer en que partes del mundo es donde se venden más productos y cuales son deficientes.

# Herramientas utilizadas
- Phyton
- Google Colab
- Power BI
